✨ CORE COMPONENTS
═══════════════════════════════════════════════════════════════════════════

  1. LoaderError (10 variants)
     ├─ ElfLoadFailed — File not found or unreadable
     ├─ ProgramNotFound — Program missing in ELF
     ├─ AttachFailed — Attachment syscall failed
     ├─ InterfaceNotFound — Interface doesn't exist
     ├─ InvalidInterface — Invalid interface name
     ├─ TcError — Traffic control error
     ├─ XdpError — XDP attachment failed
     ├─ Io — File I/O errors
     ├─ NotInitialized — Loader not initialized
     └─ AlreadyLoaded — Program already registered

  2. TcDirection (Enum)
     ├─ Ingress — Traffic control ingress
     └─ Egress — Traffic control egress

  3. XdpMode (Enum)
     ├─ Native — XDP native mode
     └─ Skb — XDP SKB fallback

  4. ElfObject (Struct)
     ├─ name — Program name
     ├─ path — File system path
     ├─ checksum — Optional SHA256/MD5
     └─ Methods: new(), with_checksum(), exists()

  5. AttachmentPoint (Struct)
     ├─ interface — Network interface (eth0, etc.)
     ├─ program_name — eBPF program name
     ├─ direction — TC direction (ingress/egress)
     ├─ priority — Filter priority (0-65535)
     └─ Methods: new(), with_priority()

  6. ProgramMetadata (Struct)
     ├─ name — Program name
     ├─ object_name — ELF object name
     ├─ interface — Attached interface (if any)
     ├─ direction — Attachment direction (if TC)
     ├─ is_xdp — True if XDP program
     └─ checksum — Optional checksum

  7. ProgramCache (Internal)
     ├─ programs: DashMap<String, ProgramMetadata>
     ├─ attachments: DashMap<String, Vec<String>>
     └─ Methods: register(), record_attachment(), get_attachments(), etc.

  8. DatapathLoader (Main API)
     ├─ new(bpf_dir, state_dir) — Create loader
     ├─ register_elf_object(obj) — Register ELF file
     ├─ register_standard_objects() — Register default Cilium objects
     ├─ load_all() — Load all registered ELF objects
     ├─ attach_tc_program(iface, prog, direction) — Attach TC program
     ├─ attach_xdp_program(iface, prog, mode) — Attach XDP program
     ├─ detach_tc_program(iface, direction) — Detach TC program
     ├─ detach_xdp_program(iface) — Detach XDP program
     ├─ get_attachments(iface, direction) — List attached programs
     ├─ get_program(name) — Lookup program metadata
     ├─ initialize_once() — One-time initialization
     ├─ is_initialized() — Check initialization state
     ├─ bpf_dir() — Get BPF directory
     └─ state_dir() — Get state directory
